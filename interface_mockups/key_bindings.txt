┌─ Agent Tree Interface - Key Bindings ──────────────────────────────────┐
│                                                                         │
│ NAVIGATION                                                              │
│ ├─ ↑/k              Move cursor up                                      │
│ ├─ ↓/j              Move cursor down                                    │
│ ├─ ←/h              Collapse current branch                             │
│ ├─ →/l              Expand current branch                               │
│ ├─ Home             Go to top of tree                                   │
│ ├─ End              Go to bottom of tree                                │
│ ├─ /                Search for step by description                      │
│ └─ n                Go to next search result                            │
│                                                                         │
│ STEP ACTIONS                                                            │
│ ├─ e                Edit selected step                                  │
│ ├─ space            Execute/pause selected step                         │
│ ├─ enter            View detailed information                           │
│ ├─ i                Interrupt running step                              │
│ ├─ r                Retry failed step                                   │
│ ├─ delete           Delete step (with confirmation)                     │
│ └─ ctrl+d           Duplicate step                                      │
│                                                                         │
│ TREE MANAGEMENT                                                         │
│ ├─ +                Add new step after current                          │
│ ├─ ctrl+n           Add new step before current                         │
│ ├─ ctrl+t           Add subtask under current step                      │
│ ├─ m                Move step (enter move mode)                         │
│ ├─ ctrl+up          Move step up in current group                       │
│ ├─ ctrl+down        Move step down in current group                     │
│ └─ ctrl+r           Refresh tree from workspace                         │
│                                                                         │
│ VIEW OPTIONS                                                            │
│ ├─ v                Toggle compact/detailed view                        │
│ ├─ f                Toggle focus mode (hide completed steps)            │
│ ├─ a                Show/hide all status indicators                     │
│ ├─ t                Toggle timing information                           │
│ └─ ctrl+l           View logs for selected step                         │
│                                                                         │
│ GLOBAL                                                                  │
│ ├─ q                Quit interface                                      │
│ ├─ ?                Show this help                                      │
│ ├─ ctrl+s           Save tree state                                     │
│ ├─ ctrl+o           Open different task tree                            │
│ └─ F5               Force refresh all agents                            │
│                                                                         │
│ STATUS INDICATORS                                                       │
│ ├─ ○                Available/pending step                              │
│ ├─ ◑                Running step                                        │
│ ├─ ●                Completed step                                      │
│ ├─ ✗                Failed step                                         │
│ └─ ⏸                Paused step                                         │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
